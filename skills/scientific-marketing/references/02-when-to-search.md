# When To Search

## Core Position

Default to the embedded knowledge base. Search is a freshness tool, not the source of the method.

Search should answer a current-fact question. It should not replace the suite's judgment.

## Search Only When

- The user asks for current, latest, recent, today, market price, current platform rule, current competitor move, or current regulation.
- The answer depends on platform behavior that changes: Google Ads, Meta, TikTok, Amazon Ads, retail media networks, AI Overviews, Perplexity, ChatGPT browsing behavior.
- The task asks for current competitors, product specs, pricing, market share, rankings, regulations, or news.
- The task is an AI visibility audit and requires current answer engine output.
- A claim needs verification before it enters a reference or public-facing deliverable.
- A high-stakes decision depends on a fact that may have changed.

## Do Not Search When

- The question is about stable marketing frameworks.
- The module already contains the needed method.
- Search would only return vendor blogs with thin evidence.
- The user asks for our point of view or a draft framework.

## Execution

When search is required:

1. State the fact that needs freshness.
2. Search primary sources first.
3. Use industry reports only when primary sources are unavailable or insufficient.
4. Treat vendor blogs as weak evidence unless they expose method and data.
5. Cite sources and dates where available.
6. Bring the result back into the suite's method.

Do not outsource reasoning to search results.

## Source Preference

- Primary: platform docs, company filings, official product pages, regulator pages, academic papers, public datasets.
- Secondary: analyst reports, industry reports, trade publications with method.
- Weak: vendor blogs, tool pages, unsourced benchmark posts, social threads.

## Outputs

If search was used, include source links and dates where available.

If search was not used, say the answer is based on the embedded method when the user may expect current facts.

## Language Notes

Do not write "I found that" unless the searched evidence changed the answer.

Prefer "current source check" over "research" when the search only verifies a fact.
