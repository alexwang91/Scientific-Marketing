## How To Review A Reference

For each reference, review in this order:

1. Core position.
2. Terms we keep.
3. Terms we ban or translate.
4. Execution steps.
5. Validation rules.
6. Failure modes.
7. Final wording pass.

## Done Criteria

A reference is ready when a future agent can use it without asking for our intent, and when every important term either names a method or tells someone what to do.
