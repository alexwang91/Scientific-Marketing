# 43 Source Authority Map

## Core Position

AI answers often reflect the sources they trust. Visibility work needs source strategy, not only website edits.

## Use When

Use this when AI answers cite competitors, outdated reviews, thin listicles, or no source that supports the brand.

## Method

1. Capture citations and implied sources.
2. Classify each source: owned, earned, retailer, review, forum, database, press, competitor.
3. Identify missing proof sources.
4. Decide which sources the brand can influence.
5. Build content, PR, review, and partner actions.

## Outputs

- Source influence table.
- Missing proof map.
- Source action plan.

## Validation

Monitor whether answer engines cite better sources and describe the brand with the intended proof.

## Failure Modes

- Publishing more owned content while third-party sources dominate.
- Ignoring old reviews.
- Treating all citations as equal.

## Language Notes

Use "source influence" instead of "authority hacking."
