# 42 Answer Engine Audit

## Core Position

AI visibility audits need query discipline. Random prompts produce random conclusions.

## Use When

Use this for brand visibility, competitor comparison, product recommendation, category entry, and reputation checks.

## Query Types

- Best product for scenario.
- Brand vs competitor.
- Is product worth it.
- Product for budget.
- Product for use case.
- Safety, privacy, reliability, or support question.
- Alternative to competitor.

## Method

1. Build 20 to 50 queries by buying situation.
2. Run them across selected answer engines.
3. Capture raw answers, sources, and rank position.
4. Tag answer quality.
5. Recommend source and content fixes.

## Outputs

- Query bank.
- Answer capture sheet.
- Visibility score.
- Fix backlog.

## Validation

Rerun with the same query set after fixes. Do not judge from one query.

## Failure Modes

- Prompting the AI with the brand name when measuring discovery.
- Ignoring negative or missing mentions.
- Failing to capture citations.

## Language Notes

Say "query set" instead of "prompt list" when measuring market discovery.
