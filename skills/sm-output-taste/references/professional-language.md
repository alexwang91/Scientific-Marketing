# Professional Language

## Core Position

Professional vocabulary is welcome when it names a method, metric, model, or decision. Empty business language is not.

## Keep

STP, 4P, 5C, AARRR, JTBD, GTM, CRM, RFM, CAC/LTV, mental availability, physical availability, category entry point, message house, brand code, uplift, CATE, HTE, holdout, geo test, MMM, incrementality, response curve, adstock, saturation, DAG, choice architecture, loss aversion, default effect, contextual bandit, next-best-treatment.

## Cut Or Translate

- Empower: say what the person can now do.
- Closed loop: name who records what and who changes what.
- Precision reach: name the segment, treatment, and channel.
- Thousand-person thousand-face: say the system chooses different treatments by context.
- Data-driven growth: say which data prevents which wrong decision.
- Growth flywheel: name the behavior that creates the next behavior.
- Ecosystem synergy: name the products, channels, and shared task.
- AI-native: name the step AI performs.

## Test

A term can stay if it helps someone execute, measure, choose, or reject an action.

## Failure Modes

- Piling terms without a decision.
- Using English acronyms to hide weak thinking.
- Replacing proof with sophistication.
