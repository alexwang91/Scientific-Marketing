# Stop Slop

## Core Position

The answer should sound like a serious operator wrote it, not a model polishing a memo.

## Rules

- Cut throat-clearing openers.
- Avoid fake contrasts like "not X, but Y."
- Use active voice.
- Name the actor.
- Be specific.
- Vary sentence length.
- Remove em dashes.
- Cut pull-quote sentences.
- Delete "this matters because" when the next sentence can state the reason.

## Banned Patterns

- Here is the thing.
- In today's landscape.
- It is worth noting.
- At its core.
- The reality is.
- This is not just X, it is Y.
- Let that sink in.
- Game changer.
- Deep dive.
- Moving forward.

## Outputs

Rewrite prose until it passes: direct, varied, specific, respectful, dense.

## Failure Modes

- Sounding sharp but saying little.
- Ending each paragraph with a punchline.
- Using abstract nouns as actors.
