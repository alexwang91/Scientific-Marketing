# Huawei Language

## Core Position

Huawei-style business language can stay when it points to operating discipline. It should not become decorative seriousness.

## Usable Terms

Customer-centric, end-to-end, frontline, campaign, battle map, operating review, metric-driven, opportunity point, value realization, scenario-led, experience loop, iron triangle, LTC, IPD, DSTE, strategy to execution, let people who hear the炮声 make judgments.

## Translation Rules

- End-to-end: map user steps, owners, metrics, and data sources.
- Campaign or battle: define phases, goal, owner, and decision cadence.
- Frontline insight: collect raw language from sales, service, retail, social, and partners.
- Metric-driven: name the metric and the decision it triggers.
- Scenario-led: name the situation, user task, and product behavior.
- Experience loop: name the touchpoint, feedback source, owner, and fix cadence.
- Iron triangle: name account, solution, delivery, and the decision they own.

## Examples

Bad: Build an end-to-end marketing loop.

Good: Map ad view, product page, consultation, purchase, and repurchase. Assign one owner, metric, and data source to each step.

Bad: Strengthen frontline insight.

Good: Each week, collect 20 raw quotes from sales, support, retail, and social. Tag them as trigger, barrier, competitor substitute, price objection, or proof need.

## Failure Modes

- Using battle language for work with no owner.
- Saying "customer-centric" without customer evidence.
- Saying "closed loop" when no one changes a decision.
