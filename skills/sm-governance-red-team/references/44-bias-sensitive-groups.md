# 44 Bias Sensitive Groups

## Core Position

Marketing systems can exploit people under stress. Segment power must come with restraint.

## Use When

Use this for children, health anxiety, financial stress, addiction risk, loneliness, elder users, and high-pressure scarcity.

## Method

1. Identify sensitive contexts.
2. Check whether targeting uses or infers vulnerability.
3. Check whether the offer increases harm.
4. Add caps, exclusions, disclosures, or human review.
5. Block the tactic if harm cannot be reduced.

## Outputs

- Sensitive group review.
- Exclusion rules.
- Safer messaging.

## Validation

A safe tactic respects user agency and does not profit from distress.

## Failure Modes

- Using anxiety as a conversion lever.
- Using scarcity on essential products.
- Treating protected or sensitive traits as targeting variables.

## Language Notes

Say "sensitive context" when risk comes from the situation, not only the person.
