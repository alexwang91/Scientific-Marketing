# 25 Agentic Marketing Ops

## Core Position

Agents can monitor, generate, compare, and remind. Humans approve money, claims, audiences, and privacy-sensitive actions.

## Use When

Use this for recurring campaign monitoring, research updates, variant generation, reporting, anomaly detection, and review workflows.

## Method

1. Define the agent task.
2. Define allowed inputs and outputs.
3. Set thresholds and escalation rules.
4. Require logs for actions and decisions.
5. Place human approval before budget, claim, audience, or data use changes.

## Outputs

- Agent workflow.
- Human approval map.
- Monitoring checklist.
- Reporting cadence.

## Validation

An agent workflow is useful when it reduces missed signals or manual work without hiding decisions.

## Failure Modes

- Letting agents optimize to platform metrics without incrementality checks.
- Letting agents change claims without legal or brand review.
- No audit trail.

## Language Notes

Say "agent drafts" or "agent flags." Say "the team approves" for decisions.
