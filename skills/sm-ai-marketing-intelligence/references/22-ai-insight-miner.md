# 22 AI Insight Miner

## Core Position

AI is strong at sorting messy language. It is weak at knowing which pattern should change strategy without business context.

## Use When

Use this for reviews, interviews, sales calls, support logs, and social comments.

## Method

1. Preserve raw quotes.
2. Tag content by trigger, pain, barrier, substitute, desired outcome, proof need, and emotion.
3. Cluster themes.
4. Estimate frequency only when the source supports it.
5. Translate themes into hypotheses and messages.

## Outputs

- Quote bank.
- Theme table.
- Message hypotheses.
- Research gaps.

## Validation

Every insight needs either a quote, a count, or a clear label as inference.

## Failure Modes

- Smoothing user language into polished slogans.
- Treating AI summaries as data.
- Ignoring minority complaints that signal a high-value segment.

## Language Notes

Use "users said" only when actual user text exists.
